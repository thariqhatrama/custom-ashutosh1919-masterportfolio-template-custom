# Masterportfolio 🚀

A customized, modern, responsive web application and showcase template. Designed using Next.js/React and optimized with Tailwind CSS.

Modified and customized by **[Thariqhat Rama](https://linkedin.com/in/thariqhatrama/)**. Original core built by [ashutosh1919](https://github.com/ashutosh1919/masterPortfolio).

## ✨ Features
- Fully responsive design.
- Highly optimized components.
- Easily customizable configuration.
- Styled with modern CSS utilities (Tailwind CSS/Bootstrap).

## 🛠️ Quick Start

Follow these steps to set up the project locally:

```bash
# Clone the repository
git clone https://github.com/thariqhatrama/custom-masterportfolio-portfolio.git

# Navigate into the project folder
cd custom-masterportfolio-portfolio

# Install dependencies and start the development server
npm run build
npm install
```
